# Privacy

Clean Reader is designed to be local-first.

## What Clean Reader Collects

Clean Reader does not collect, sell, transmit, or share personal data.

## What Stays Local

Clean Reader stores only reader preferences locally in your browser, such as:

- Selected font
- Custom local font name
- Text size
- Line spacing
- Column width
- Theme

These preferences are saved with `chrome.storage.local`.

## Page Access

Clean Reader reads the active page only when you click the extension toolbar button. It uses that temporary access to extract readable article content and render it in an overlay.

## Network Access

Clean Reader does not send article text, URLs, browsing history, preferences, or other data to a remote server. It does not load remote code.

## Contact

Use the repository issue tracker for privacy questions or bug reports.
