# Security

Clean Reader is a browser extension that runs locally in the active tab after the user clicks the extension button.

## Reporting Security Issues

Please do not open a public issue for a sensitive security problem. If this repository does not list a private security contact, open a minimal issue asking for a secure contact path without disclosing details.

## Security Expectations

- No remote code
- No analytics
- No tracking
- Minimal permissions
- Local-only preference storage

## Supported Versions

This project is early-stage. Security fixes should target the latest version on the main branch.
