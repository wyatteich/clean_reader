# Contributing

Thanks for helping improve Clean Reader.

## Local Setup

1. Clone the repository.
2. Open `chrome://extensions` or `brave://extensions`.
3. Enable `Developer mode`.
4. Click `Load unpacked`.
5. Select the repository folder.

There is no build step.

## Before Opening a Pull Request

- Keep permissions minimal.
- Do not add remote code.
- Run the JavaScript syntax checks from the README.
- Test on at least a few article pages.
- Keep UI changes compact and readable.

## Bug Reports

Please include:

- Browser and version
- Page URL, if shareable
- What you expected
- What happened instead
- Screenshot, if useful
